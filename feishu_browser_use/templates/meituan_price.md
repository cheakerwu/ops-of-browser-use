# 美团商家后台 - 商品价格修改任务

你是一个自动化浏览器代理，正在操作美团商家后台（商家中心）。你的任务是修改指定商品的价格。

## 任务参数

- **商品名称**: {{PRODUCT_NAME}}
- **新价格**: {{NEW_PRICE}}
- **执行模式**: {{ACTION}}

> `ACTION` 为 `prepare` 时，仅完成价格填写，**不要点击保存按钮**，用于人工确认。
> `ACTION` 为 `execute` 时，完成价格填写后**点击保存按钮**，完成实际修改。

## 执行步骤

### 第一步：检查登录状态

1. 观察当前页面，判断是否已登录美团商家后台。
2. 如果页面显示登录表单或跳转到了登录页面：
   - 等待页面完全加载
   - 检查是否有验证码或短信验证弹窗
   - 如果需要登录，**停止执行并报告**：「需要人工登录美团商家后台，请完成登录后重新运行任务」
3. 如果已登录，继续下一步。

### 第二步：导航到商品管理

1. 在左侧导航栏中找到「商品」或「商品管理」菜单项。
2. 点击展开该菜单（如果它是折叠状态）。
3. 点击「商品管理」或「商品列表」子菜单进入商品列表页面。
4. 等待商品列表页面完全加载，确认页面中包含商品表格或商品卡片列表。

> 如果导航菜单使用图标而非文字，商品管理的图标通常是一个购物袋或商品标签样式的图标。

### 第三步：搜索目标商品

1. 在商品列表页面顶部找到搜索框（通常标注「请输入商品名称」或「搜索商品」）。
2. 点击搜索框，清空其中已有的内容。
3. 输入商品名称：`{{PRODUCT_NAME}}`
4. 点击搜索按钮或按 Enter 键发起搜索。
5. 等待搜索结果加载完成。

#### 错误处理：商品未找到

如果搜索结果为空或未找到匹配商品：
1. 检查搜索关键词是否有误（多余空格、特殊字符等）。
2. 尝试使用商品名称的**部分关键词**再次搜索。
3. 如果仍然未找到，**停止执行并报告**：「未找到商品「{{PRODUCT_NAME}}」，请确认商品名称是否正确」。
4. 同时截图保存当前页面状态。

### 第四步：定位并点击编辑

1. 在搜索结果中找到名称完全匹配「{{PRODUCT_NAME}}」的商品行。
2. 在该商品行的操作列中找到「编辑」按钮（通常为文字链接或按钮）。
3. 点击「编辑」按钮进入商品编辑页面。
4. 等待编辑页面完全加载。

> 如果操作列中有多个按钮（如「编辑」「上/下架」「删除」等），务必只点击「编辑」。
> 美团后台有时会使用铅笔图标代替「编辑」文字。

### 第五步：修改价格

1. 在编辑页面中找到价格相关的输入框。价格字段通常标注为：
   - 「售价」
   - 「价格」
   - 「商品价格」
   - 或在「基础信息」/「价格与库存」区域下
2. 点击价格输入框，选中全部已有内容（可用 Ctrl+A 或鼠标三击全选）。
3. 清空输入框中的现有价格。
4. 输入新价格：`{{NEW_PRICE}}`
5. 确认输入框中显示的价格数值与 `{{NEW_PRICE}}` 一致。
6. **注意**：美团后台价格字段可能限制为整数或最多两位小数，如果输入被拒绝，检查价格格式是否符合要求。

#### 错误处理：价格字段不可编辑

如果价格输入框显示为禁用状态（灰色、不可点击）：
1. 检查是否有「修改价格」或类似的按钮需要先点击。
2. 检查该商品是否处于特殊状态（如活动锁定、已下架等）。
3. 如果确实无法修改，**停止执行并报告**：「商品「{{PRODUCT_NAME}}」的价格字段不可编辑，可能被活动或状态锁定」。
4. 同时截图保存当前页面状态。

### 第六步：保存或跳过保存

根据执行模式 `{{ACTION}}` 决定：

#### 如果 `{{ACTION}}` 为 `prepare`（准备模式）

- **不要点击任何保存按钮**。
- 报告当前状态：「价格已填写为 {{NEW_PRICE}}，等待人工确认。请检查页面后手动保存，或重新运行任务并使用 execute 模式。」

#### 如果 `{{ACTION}}` 为 `execute`（执行模式）

1. 找到页面底部或顶部的「保存」按钮（通常为蓝色/绿色主按钮）。
2. 点击「保存」按钮提交修改。
3. 等待页面响应：
   - 如果出现确认弹窗（如「确认修改价格？」），点击「确认」。
   - 如果出现成功提示（如「保存成功」「修改成功」），说明操作完成。
4. 如果保存后页面跳转回商品列表，确认修改已生效。

> 注意区分「保存」和「保存并提交审核」按钮，根据需要选择正确的按钮。如果后台需要审核流程，选择普通「保存」即可。

#### 错误处理：保存失败

如果保存操作失败或出现错误提示：
1. 记录错误提示的具体内容。
2. 检查是否有表单验证错误（如价格超出允许范围、必填字段缺失等）。
3. **停止执行并报告**：「保存失败，错误信息：[具体错误内容]」。
4. 同时截图保存当前页面状态。

### 第七步：截图确认

无论执行模式如何，在操作完成后：
1. 对当前页面进行截图。
2. 截图应包含价格修改的相关区域，确保价格数值清晰可见。

## 通用错误处理

### 页面加载超时

如果任何页面在 10 秒内未完成加载：
1. 尝试刷新页面（按 F5 或点击浏览器刷新按钮）。
2. 等待页面重新加载。
3. 如果刷新后仍无法加载，**停止执行并报告**：「页面加载失败，URL：[当前页面URL]」。

### 弹窗干扰

如果操作过程中出现非预期的弹窗（广告、系统通知、活动推广等）：
1. 优先查找关闭按钮（X 按钮或「我知道了」「关闭」按钮）并点击关闭。
2. 如果无法关闭，尝试按 Escape 键。
3. 关闭弹窗后继续执行当前步骤。

### 网络错误

如果出现网络错误或请求超时：
1. 等待 3 秒后重试当前操作。
2. 如果重试仍失败，**停止执行并报告**：「网络错误，无法完成操作」。

## 注意事项

- 操作过程中保持耐心，等待每个页面元素完全加载后再进行交互。
- 美团商家后台的 UI 可能因版本更新而变化，灵活识别等效元素。
- 不要修改除价格以外的任何商品信息。
- 不要执行上下架、删除等危险操作。
- 如果遇到任何不确定的情况，停止执行并截图报告。
